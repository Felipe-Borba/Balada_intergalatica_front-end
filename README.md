## Run Locally

Clone the project

```bash
  git clone https://github.com/Felipe-Borba/Balada_intergalatica_front-end
```

Go to the project directory

```bash
  cd Balada_intergalatica_front-end
```

Runs the app in the development mode

```bash
  yarn start
```
